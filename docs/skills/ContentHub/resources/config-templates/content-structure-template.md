#### 可选模块
1. **配图**: 为文章添加配图
   - 适用条件: 所有文章（建议必选）
   - 内容要求:
     - 每篇文章至少1-3张配图
     - 图片必须下载到本地: `{账号}/output/{日期}/images/`
     - 文章中使用相对路径: `./images/{文件名}`
     - 创建图片清单: `images/manifest.txt`
     - 标注图片来源: `*图片来源: AI生成*` 或 `*图片来源: XX平台*`

2. **预告/连载**: 内容规划与预告
   - 适用条件: 有明确的内容规划或连载计划时
   - 内容要求:
     - 预告下期主题、连载内容预告
     - **重要**: 必须在 `{账号}/output/content-plan.md` 中记录所有预告和连载计划
     - 生成文章前先查看 `content-plan.md`，确保能够兑现
     - 如无法兑现，则更新计划，删除或调整预告
   - content-plan.md 示例格式:
     ```markdown
     # 内容规划

     ## 已发布的预告
     - [2026-01-13] 周二将推出"一人食指南" ✅ 已发布

     ## 待兑现的预告
     - [2026-01-14] 周三将推出"外卖精选"
     - [2026-01-15] 周四将推出"轻食推荐"

     ## 连载计划
     - 【系列】网红餐厅测评系列（共10期）
       - 第1期: 火烧云傣家菜 ✅ 已发布
       - 第2期: XX餐厅 ⏳ 计划中
       - 第3期: XX餐厅 ⏳ 计划中
     ```

## 通用结构规则
- 标题格式: 使用 # ## ### 表示层级
- 段落规则: 每段不超过200字，段间空一行
- 引用格式: 使用 > 引用餐厅官方介绍或网友评价
- 图片格式: `
![图片描述](./images/{文件名})
`
- 信息标注格式: 💰价格、📍地址、⏰营业时间、⚠️注意事项
- 禁止事项:
  - ❌ 禁止在文章中直接使用外部图片URL（必须下载到本地）
  - ❌ 禁止使用未经标注的网络图片
