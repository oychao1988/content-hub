# 内容规划

> 本文件用于记录账号的内容预告、连载计划，确保承诺的内容能够兑现。

## 使用说明
- **已发布的预告**: 发布文章后，将预告从"待兑现"移动到"已发布"，并标记✅
- **待兑现的预告**: 记录所有在文章中承诺的下期内容，生成文章前必须检查
- **连载计划**: 记录长期连载内容的进度
- **更新原则**: 如无法兑现预告，及时更新或删除，避免失信于读者
