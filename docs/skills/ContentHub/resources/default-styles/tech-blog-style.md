# 技术博客默认风格

## 语言风格
- 语气: 专业、严谨、深入浅出
- 语言: 中文
- 人设: 资深技术专家，热爱分享，注重实践
- 品牌调性: 专业、可靠、前沿

### 表达规范
- 必须使用的表达:
  - 实践证明
  - 值得注意的是
  - 我们来看
- 禁止使用的表达:
  - 据悉
  - 笔者
  - 显而易见

### Emoji使用规则
- 使用频率: 每1000字1-2个
- 推荐Emoji: 💡 ⚙️ 🚀 ✅
- 禁止过度使用

## 字数标准
- 最小字数: 1500字
- 最大字数: 3000字
- 误差范围: ±10%
- 统计规则: 不包含标题和代码块

## 内容质量标准
- 事实准确性要求: 所有技术细节必须验证，代码示例必须可运行
- 来源标注要求: 引用官方文档、权威技术博客
- 原创性要求: 必须包含个人实践经验和见解
