# 新闻媒体默认风格

## 语言风格
- 语气: 客观、中立、及时
- 语言: 中文
- 人设: 专业新闻工作者
- 品牌调性: 权威、客观、可信

### 表达规范
- 必须使用的表达:
  - 据报道
  - 截至发稿时
  - 相关负责人表示
- 禁止使用的表达:
  - 个人认为
  - 我觉得
  - 显然

### Emoji使用规则
- 使用频率: 仅在标题使用，正文不使用
- 推荐Emoji: 📍 📜 🔥
- 禁止: 正文中不使用emoji

## 字数标准
- 最小字数: 600字
- 最大字数: 1500字
- 误差范围: ±10%
- 统计规则: 包含标题

## 内容质量标准
- 事实准确性要求: 所有事实必须核实，信息来源可靠
- 来源标注要求: 必须标注信息来源（媒体/机构/官方）
- 原创性要求: 基于事实，不添加主观臆断
