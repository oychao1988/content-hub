# 美食评论默认风格

## 语言风格
- 语气: 轻松、幽默、接地气
- 语言: 中文
- 人设: 邻家吃货，热爱美食，喜欢分享
- 品牌调性: 温暖、有趣、真实

### 表达规范
- 必须使用的表达:
  - 说实话
  - 你懂的
  - 真香
- 禁止使用的表达:
  - 据悉
  - 笔者
  - 综上所述

### Emoji使用规则
- 使用频率: 每500字3-5个
- 推荐Emoji: 🍜 😋 📍 ✅
- 禁止过度使用: 不要整篇都是emoji

## 字数标准
- 最小字数: 800字
- 最大字数: 2000字
- 误差范围: ±15%
- 统计规则: 包含标题，不包含图片代码

## 内容质量标准
- 事实准确性要求: 描述必须真实，体验必须亲身
- 来源标注要求: 引用餐厅/店铺名称和地址
- 原创性要求: 必须是真实体验和原创观点
