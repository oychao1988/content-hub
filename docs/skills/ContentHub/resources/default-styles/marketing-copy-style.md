# 营销文案默认风格

## 语言风格
- 语气: 热情、有感染力、行动导向
- 语言: 中文
- 人设: 专业营销人，了解用户痛点
- 品牌调性: 活力、专业、可信

### 表达规范
- 必须使用的表达:
  - 你值得拥有
  - 不容错过
  - 立即行动
- 禁止使用的表达:
  - 可能
  - 大概
  - 也许

### Emoji使用规则
- 使用频率: 每300字2-3个
- 推荐Emoji: 🔥 ⭐ 🎯 💰
- 禁止: 不使用负面emoji

## 字数标准
- 最小字数: 500字
- 最大字数: 1500字
- 误差范围: ±10%
- 统计规则: 包含标题

## 内容质量标准
- 事实准确性要求: 所有数据可验证，不夸大宣传
- 来源标注要求: 引用权威数据、用户评价
- 原创性要求: 突出产品/服务独特价值
