# 版本历史

## [2.0.0] - 2026-01-13

### 重大更新

#### 配图系统升级
- **强制本地下载**: 所有图片必须下载到本地，禁止在文章中使用外部URL
- **图片清单文件**: 每篇文章必须创建 `images/manifest.txt`，记录图片来源和用途
- **相对路径引用**: 文章中使用 `./images/{文件名}` 引用图片
- **命名规范**: 统一图片命名格式 `YYYY-MM-DD-{板块简称}-{序号}.jpg`

#### 内容规划系统
- **content-plan.md**: 新建内容规划模板，管理预告和连载计划
- **预告管理**: 从"完全禁止"改为"规范管理"，通过content-plan.md记录和追踪
- **连载支持**: 支持连载系列进度追踪和规划
- **兑现检查**: 生成文章前检查待兑现的预告，确保能够承诺

#### 时效性提升
- **时效性要求**: 优先使用最近7-30天的网络信息
- **有效性确认**: 时效性强的内容（新开业、优惠）必须确认仍在有效期内
- **来源区分**: 明确区分"官方信息"和"网友评价/媒体报道"

#### 发布功能
- **发布模块**: 新增内容发布模块（可选）
- **平台支持**: 支持微信公众号等平台自动发布
- **发布配置**: 在account-config.md中配置发布平台信息
- **发布前检查**: 支持发布前的自动检查（图片、格式等）

### 配置模板更新
- **writing-style-template.md**: 添加时效性、配图下载、禁止事项要求
- **content-structure-template.md**: 添加配图模块、预告/连载管理模块
- **account-config-template.md**: 添加发布平台配置字段
- **content-plan-template.md**: 新建内容规划模板文件

### 文档优化
- **SKILL.md**: 添加MCP工具配置、发布流程说明
- **模板文件组织**: topic-history-template.md移至config-templates/目录
- **TEMPLATE-UPDATE-SUMMARY.md**: 新建模板更新总结文档

### 质量保证增强
- **配图质量门禁**: 检查图片是否下载、文件是否正常、来源是否标注
- **时效性检查**: 质量检查中增加时效性验证
- **图片清单检查**: 验证manifest.txt文件存在且完整

## [1.0.0] - 2026-01-13

### 新增
- 初始版本发布
- 支持多账号管理
- 自动创建账号配置
- 模块化工作流（选题/生成/配图/质检）
- 默认风格库（技术博客、营销文案、美食评论、新闻媒体）
- 完整的用户指南和工作流文档

### 特性
- 每个账号有独立的4个配置文件
- 灵活的内容板块规划
- 自动配图生成（豆包AI）
- 质量门禁检查
- 选题历史记录
