## 维护指南

### 添加新设计文档

1. 创建新文件：`<功能名>-design.md`
2. 使用 [templates](../references/TEMPLATES.md) 中的设计文档模板
3. 在上表中添加文档条目
4. 设置初始状态为 ❌

### 更新状态

当功能实施完成后：
1. 将状态更新为 ✅
2. 添加实施时间
3. 可选：将文档移到其他对应目录
