- **详细描述**:
  - 读取所有页面组件代码
  - 分析每个页面的：
    - 输入表单和交互元素
    - 按钮操作和触发条件
    - API 调用和数据处理
    - 页面导航和跳转逻辑
  - 绘制页面跳转关系图
  - 生成交互逻辑分析报告
- **完成标准**: 生成完整的页面交互逻辑分析文档
- **执行结果**: 已生成 FRONTEND-ANALYSIS.md，包含：

  - 表单验证规则总结（9 个类别）
