## 维护指南

### 添加新开发文档

1. 创建新文件：`<功能名>-PLAN.md` 或 `<功能名>-SUMMARY.md`
2. 使用 [templates](../references/TEMPLATES.md) 中的相应模板
3. 在上表中添加文档条目

### 文档生命周期

- **活跃文档**: 保留在 `development/`
- **完成文档**: 移到 `archive/phases/` 或 `archive/sessions/`
